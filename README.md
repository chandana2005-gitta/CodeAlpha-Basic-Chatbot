# Basic Chatbot

## Overview

Basic Chatbot is a simple rule-based chatbot developed in Python. It interacts with users through predefined responses and demonstrates the fundamentals of conditional statements and user interaction.

## Features

* Responds to greetings
* Answers basic questions
* Simple text-based conversation
* Exit command using "bye"
* Beginner-friendly Python project

## Technologies Used

* Python

## How to Run

1. Install Python on your system.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run:

```bash
python chatbot.py
```

## Sample Interaction

```text
🤖 ChatBot: Hello! Type 'bye' to exit.

You: hello
Bot: Hi!

You: how are you
Bot: I'm fine, thanks!

You: bye
Bot: Goodbye!
```
